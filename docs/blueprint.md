# **App Name**: MinimalFolio

## Core Features:

- Project Gallery: Display featured projects with titles and brief descriptions, which link to dedicated project detail pages.
- Project Details Pages: Showcase detailed information for each project using Markdown, including descriptions, images, and links.
- Blog Feed: Present a chronological list of blog posts with clickable titles and short excerpts.
- Blog Post Pages: Dedicated pages for each blog post, rendered from Markdown files.
- About Me Section: Include a professional summary and relevant background information.
- Responsive Navigation: Ensure seamless navigation across all devices with a minimalist design.

## Style Guidelines:

- Primary color: Soft slate blue (#778DA9), for a professional, technical feel without being cold. In HSL, the hue is close to blue, but the low saturation avoids unwanted connotations.
- Background color: Light grey (#E0E1DD), creating a near-white, non-glaring canvas with a subtle connection to the primary.
- Accent color: Muted stone blue (#415A77) - analogous to the primary but darker and less saturated - for interactive elements to draw focus without overwhelming the design.
- Headline font: 'Space Grotesk' sans-serif for headlines and other short pieces of text. Body text: 'Inter' sans-serif for readability in longer blocks.
- Use a minimalist, grid-based layout to provide structure without clutter, emphasizing white space and readability.
- Employ simple line icons for navigation and social media links, in the stone blue accent color, ensuring they are both functional and subtle.
