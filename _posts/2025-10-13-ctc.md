**Code & Text Cleaner Pro** adalah *offline toolbox* serbaguna berbasis web yang berfungsi untuk **membersihkan, mengonversi, dan mengelola teks serta kode** (HTML, CSS, JS) dengan cepat dan mudah — langsung dari browser, tanpa perlu koneksi internet.  

---

## ✨ Fitur Utama

### 📝 Text Tools  
> Editor ringan untuk memproses teks dan format dasar.

- Mengubah huruf menjadi **lowercase**, **UPPERCASE**, atau **Title Case**  
- Menghapus spasi ganda dan baris kosong  
- *Prettify JSON* (auto indentasi dan format)  
- Hitung otomatis **jumlah kata** dan **karakter**  
- Menyimpan, memuat, dan mereset data via `localStorage`  
- Mendukung drag & drop file `.txt` atau `.md`  

### 💻 Code Cleaner  
> Membersihkan dan merapikan file HTML, CSS, serta JavaScript.

- *Minify HTML*, *Clean CSS*, dan *Clean JS*  
- Hapus komentar kode otomatis  
- Format ulang HTML dasar (*pretty print*)  
- Live preview dalam iframe sandbox  
- Fitur ekspor dan salin hasil  

### 🔡 HTML Decoder  
> Ubah karakter HTML Entities menjadi teks normal, dan sebaliknya.

- Encode / Decode manual  
- Auto detect & decode otomatis  
- Copy hasil hanya dengan satu klik  

### 🔀 Combine Tool  
> Gabungkan HTML, CSS, dan JS menjadi satu file HTML utuh.

- Kolom terpisah untuk HTML / CSS / JS  
- Hasil otomatis digabung ke `<style>` & `<script>`  
- Preview, copy, dan download hasil `.html`  

### ✂️ Split Tool  
> Pisahkan file HTML yang mengandung `<style>` dan `<script>`.

- Pisah otomatis menjadi tiga file: HTML, CSS, JS  
- Download / copy masing-masing hasil  
- Cocok untuk refactor cepat

## 🎨 Desain dan UI

Mengusung tema **Glassmorphism Modern**, dengan elemen:
- Warna dasar lembut `#f5f7ff` & `#f0f2ff`  
- Aksen ungu & indigo (`#6366f1`, `#a855f7`)  
- Bayangan lembut & *blur* transparan  
- Animasi *fade*, *slide*, dan *hover* interaktif  
- Responsif untuk semua ukuran layar  

---

## ⚙️ Struktur Utama

index.html   → Struktur & layout utama
style.css    → (Inline / external) styling glassmorphism
sc.js        → Script utama (tab nav, transform, combine, split)

---

## 📦 Cara Menggunakan

1. Buka file `index.html` di browser (bisa **offline**)  
2. Pilih tab sesuai kebutuhan: **Text / Code / Decoder / Combine / Split**  
3. Paste teks langsung atau drag & drop file  
4. Klik tombol aksi (misal: `Minify`, `Decode`, `Split`)  
5. Simpan, salin, atau unduh hasilnya  

---

## 💾 Fitur Penyimpanan Otomatis

> Setiap 5 detik, data teks otomatis tersimpan ke `localStorage`.  
Jika browser ditutup atau direfresh, data terakhir akan termuat kembali secara otomatis.
