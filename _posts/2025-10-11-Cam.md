# 🎥 AV Test Studio Pro - Documentation

## 🧩 Overview

**AV Test Studio Pro** is a professional audio-visual testing and diagnostics tool designed to help users test their **camera and microphone functionality** directly in a web browser.  
The app provides a full-featured interface for testing, recording, and analyzing audio-visual performance.

## ⚙️ Features

### 🎥 Camera Testing
- Real-time **camera preview**
- **Device selection** between multiple cameras
- Adjustable **video quality**
- **Zoom control** (digital)
- **Snapshot capture** for still images
- **Video recording** with audio
- Full **camera diagnostics**: resolution, FPS, and device info

## 🧱 Code Structure

### 🏗 HTML Layout
Organized into modular sections:

1. **Header Section** – Title, branding, and description  
2. **Main Grid** – Two-column responsive layout for camera & audio panels  
3. **Camera Panel** – Preview, device selector, controls, diagnostics  
4. **Audio Panel** – Visualization, mic controls, test options  
5. **Diagnostics Panel** – System info cards & status  
6. **Status Bar** – Global status messages & indicators

### 🎨 CSS Styling
- **Dark mode** design with cyan & blue accents  
- **Glassmorphism** UI (blurred panels)  
- **Responsive layout** for all screen sizes  
- **Subtle animations** & hover effects  
- **Custom controls** for sliders, buttons, and selectors  
  <img width="1366" height="768" alt="Screenshot (26)" src="https://github.com/user-attachments/assets/dceab565-92f8-4f0d-8039-f7d71a7fb5a5" />

### 🧠 JavaScript Structure
Organized into a class-based controller: `AVTestStudioPro`

#### Main Methods:
- **Camera Methods**
  - `startCamera()`, `stopCamera()`
  - `switchCamera()`
  - `takeSnapshot()`
  - `startRecording()` / `stopRecording()`

- **Audio Methods**
  - `startMicrophone()`, `stopMicrophone()`
  - `toggleEcho()`, `toggleNoiseReduction()`
  - `playTestTone()`
  - `startVisualization()`

- **Diagnostics**
  - `updateDiagnostics()`
  - `runFullDiagnostics()`

- **Utility**
  - `updateTestResult()`
  - `updateStatus()`
  - `enumerateDevices()`

## 🧭 How to Use

### 🎥 Camera
1. Select camera device  
2. Click **“Start Camera”**  
3. Adjust **quality & zoom**  
4. Take **snapshots or record video**  
5. Review **camera info & test results**

### 🎙 Microphone
1. Select mic device  
2. Click **“Start Mic”**  
3. Switch between **frequency/waveform**  
4. Enable **echo** or **noise reduction**  
5. Play **test tone** for audio verification

### 💻 System Diagnostics
1. View real-time system info  
2. Run **Full Diagnostics**  
3. Review battery, memory, network, and CPU status

## 🧩 Technical Implementation

### 🔊 Media APIs
- `MediaDevices.getUserMedia()` – camera & mic access  
- `MediaRecorder` – video/audio recording  
- `Web Audio API` – visualization & analysis

### 🎛 Visualization Techniques
- **Frequency bars**  
- **Waveform (oscilloscope)**  
- **Canvas rendering** for smooth real-time drawing

### 📱 Responsive Design
- **Desktop**: Two-column layout  
- **Tablet**: Compact single-column  
- **Mobile**: Optimized controls and layout
