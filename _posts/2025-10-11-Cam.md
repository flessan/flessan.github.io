# AV Test Studio Pro - Documentation

## Overview

AV Test Studio Pro is a professional audio-visual testing and diagnostics tool designed to help users test their camera and microphone functionality directly in a web browser. The application provides a comprehensive interface for testing, recording, and analyzing audio-visual equipment performance.

## Live Preview & Privacy Policy

- **Live Preview**: [https://flessan.pages.dev/cam](https://flessan.pages.dev/cam)
- **Privacy Policy**: [https://flessan.pages.dev/cam/p.html](https://flessan.pages.dev/cam/p.html)

## Features

### Camera Testing
- **Camera Preview**: Real-time camera feed display
- **Device Selection**: Switch between available cameras
- **Quality Control**: Adjust video quality settings
- **Zoom Functionality**: Digital zoom control
- **Snapshot Capture**: Take still images from the camera feed
- **Recording**: Record video with audio
- **Camera Diagnostics**: Resolution, FPS, and device information

- **Audio Visualization**: Real-time frequency and waveform visualization
- **Device Selection**: Switch between available microphones
- **Echo Control**: Enable/disable echo with volume control
- **Noise Reduction**: Toggle noise suppression

- **Audio Diagnostics**: Input level, frequency, and device information

### JavaScript Functionality
The JavaScript code is organized into a class-based structure:

1. **AVTestStudioPro Class**
   - Main application controller
   - Manages all functionality and state

2. **Camera Methods**
   - `startCamera()`: Initialize and display camera feed
   - `stopCamera()`: Stop camera and release resources
   - `switchCamera()`: Change between available cameras
   - `takeSnapshot()`: Capture and download still image
   - `startRecording()`/`stopRecording()`: Record video with audio

3. **Audio Methods**
   - `startMicrophone()`: Initialize microphone and visualization
   - `stopMicrophone()`: Stop microphone and release resources
   - `toggleEcho()`: Enable/disable audio echo
   - `toggleNoiseReduction()`: Enable/disable noise suppression
   - `playTestTone()`: Generate 440Hz test tone
   - `startVisualization()`: Render audio visualization

4. **Diagnostics Methods**
   - `updateDiagnostics()`: Gather system information
   - `runFullDiagnostics()`: Perform comprehensive system check

5. **Utility Methods**
   - `updateTestResult()`: Update test result indicators
   - `updateStatus()`: Display status messages
   - `enumerateDevices()`: Detect available cameras and microphones

## How to Use

### Camera Testing
1. Select a camera from the dropdown menu
2. Click "Start Camera" to begin the preview
3. Adjust quality and zoom settings as needed
4. Take snapshots or start recording as desired
5. View camera information and test results in the panel

### Microphone Testing
1. Select a microphone from the dropdown menu
2. Click "Start Mic" to begin audio capture
3. Toggle between frequency and waveform visualization
4. Enable echo or noise reduction as needed
5. Play test tones and view audio information

### System Diagnostics
1. View real-time system information in the diagnostics panel
2. Click "Run Full Diagnostics" for a comprehensive system check
3. Review battery, memory, network, and CPU status

## Technical Implementation

### Media APIs
The application uses modern browser APIs for media handling:
- `MediaDevices.getUserMedia()` for camera and microphone access
- `MediaRecorder` for video recording
- `Web Audio API` for audio processing and visualization

### Visualization Techniques
- **Frequency Visualization**: Bar chart representation of audio frequencies
- **Waveform Visualization**: Oscilloscope-style display of audio waveforms
- **Canvas Rendering**: Efficient drawing using HTML5 Canvas

### Responsive Design
The interface adapts to different screen sizes:
- Desktop: Two-column layout
- Tablet: Single-column layout
- Mobile: Optimized controls and layout

## Privacy Considerations
