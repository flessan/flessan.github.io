# ⏰ Jam – Advanced Analog & Digital Clock

**Jam** is a modern **HTML, CSS, and JavaScript** application that displays a **real-time analog clock** alongside a **digital display** and multiple advanced features.  
It is fully static and can be hosted easily (e.g., on GitHub Pages).

---

## ✨ Features

- **Real-Time Analog Clock**  
  - Smooth-moving hands for hours, minutes, and seconds.  
  - Multiple clock face styles: Classic, Modern, Minimal, Roman numerals.  

- **Digital Clock & Date Display**  
  - Toggle between **12h / 24h** formats.  
  - Shows current date in a human-readable format.

- **Dynamic Themes**  
  Built-in theme switching:  
  - 🌑 Dark  
  - 🌊 Ocean  
  - 🌅 Sunset  
  - 🌳 Forest  

- **Sound Effects**  
  - Tick sounds enabled by default.  
  - Toggle sound on/off via button.  

- **Information Tabs**  
  1. **Time Info**  
     - Current Timezone  
     - UTC Offset  
     - Day of Year  
     - Week Number  
     - Day Progress (% + progress bar)  
     - Approximate Sunrise & Sunset  
  2. **World Clock**  
     - Shows real-time in: New York, London, Tokyo, Sydney  
  3. **Pomodoro Timer**  
     - 25-minute focus timer with **Start/Pause/Reset**  
     - Desktop notifications (if permitted)

- **Responsive Layout**  
  - Grid-based design adapts for mobile and desktop.  

---

## 📂 Code Structure

1. **HTML**
   - `<header>`: logo and theme switcher.  
   - `<main>`:  
     - **Clock card**: analog + digital + controls.  
     - **Tabs**: info, world clock, Pomodoro.  
   - `<footer>`: app credits.  

2. **CSS**
   - Global variables for theming.  
   - Separate style rules for each clock type and theme.  
   - Responsive grid + interactive hover effects.  

3. **JavaScript (`ClockApp` class)**
   - Handles all dynamic updates:
     - Analog hand rotation.  
     - Digital clock + date update.  
     - Theme and style switching.  
     - Sound control.  
     - Info tab updates (timezone, day progress, etc).  
     - World clock updates.  
     - Pomodoro logic (start/pause/reset).  
   - Uses `requestNotificationPermission` for Pomodoro alerts.  

---

## 🚀 Usage

1. Save the code as `index.html`.  
2. Open with any modern browser.  
3. Features load automatically:
   - Clock updates every second.  
   - Tabs and themes are interactive.  

---

## 🔧 Customization

- **Default Theme**: change `data-theme="ocean"` in `<html>`.  
- **Clock Size**: edit `--clock-size` in CSS.  
- **Pomodoro Duration**: modify `this.pomodoroTime = 25 * 60;` in JS.  
- **World Clock Cities**: edit `cities` object in `updateWorldClocks()`.  
- **Sound Effects**: tweak `playSound()` frequency/type in JS.  

---

## 📝 Notes

- This app is 100% static (no backend).  
- Sunrise & sunset are **approximate** (calculated by formula, not live API).  
- Works offline once loaded.
