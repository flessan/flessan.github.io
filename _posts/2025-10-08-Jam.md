# ⏰ Jam – Advanced Multi-Feature Clock

**Jam** is a modern **HTML, CSS, and JavaScript** application that combines an **analog clock** and **digital clock** with a full suite of productivity and time-tracking tools.  
It is 100% static and works perfectly on **GitHub Pages** or any static hosting.

---

## ✨ Features

### 🕰 Real-Time Clocks
- **Analog Clock**
  - Smooth-moving hands (hour, minute, second).
  - Multiple clock styles: Classic, Modern, Minimal, Roman numerals.  
- **Digital Clock**
  - Live 12h/24h toggle.
  - Displays full date (weekday, month, year).

### 🎨 Themes & UI
- Four built-in themes:
  - 🌑 Dark  
  - 🌊 Ocean  
  - 🌅 Sunset  
  - 🌳 Forest  
- **Custom theme option** and **Auto Theme Switching** (changes automatically by time).  
- **Fullscreen mode** & **Focus Mode** for distraction-free use.  

### 🔊 Sound Control
- Tick & action sounds (toggleable).  
- Volume slider in **Settings**.  

### 📊 Information Tabs
1. **Time Info**
   - Timezone & UTC offset  
   - Day of Year & Week Number  
   - Day Progress (with animated progress bar)  
   - Sunrise & Sunset (approximate)  
   - Location Info (Latitude, Longitude, Country, Region)  

2. **World Clock**
   - Live time in: **New York, London, Tokyo, Sydney**  

3. **Pomodoro Timer**
   - 25-minute focus sessions  
   - Start / Pause / Reset controls  
   - Notifications & sound alarms  

4. **Countdown Timer**
   - Customizable hours, minutes, seconds  
   - Start / Pause / Reset countdown  

5. **Stopwatch**
   - Start / Pause / Reset  
   - Lap recording  

6. **Alarm**
   - Add alarms with labels  
   - Manage multiple alarms  

7. **Calendar**
   - Full month view  
   - Navigate: Previous, Today, Next  

8. **Statistics**
   - Pomodoros completed  
   - Total focus time  
   - Alarms triggered  
   - Stopwatch usage count  
   - Export/Import settings  

9. **Events**
   - Add custom event countdowns  
   - Display name + date  

---

## 📂 Code Overview

- **HTML**:  
  - Layout with `<header>`, `<main>` cards, `<footer>`.  
  - Tabs for each feature (Info, World Clock, Pomodoro, Countdown, Stopwatch, Alarm, Calendar, Stats, Events).  
- **CSS**:  
  - Variables for theming.  
  - Responsive grid.  
  - Styles for clocks, controls, modals, and cards.  
- **JavaScript (`ClockApp` class)**:  
  - Manages clocks, timers, alarms, notifications, and theme logic.  
  - Stores usage statistics.  
  - Handles sound playback & settings.  

---

## 🚀 Usage

1. Save the file as `index.html`.  
2. Open in any modern browser.  
3. Explore features through the tabs and settings.  

---

## 🔧 Customization

- **Default Theme** → Change `data-theme="ocean"` in `<html>`.  
- **Clock Size** → Edit `--clock-size` in CSS.  
- **Pomodoro Duration** → Adjust `this.pomodoroTime = 25 * 60;` in JS.  
- **World Clock Cities** → Edit `cities` in `updateWorldClocks()`.  
- **Sounds** → Customize `playSound()` mappings in JS.  
- **Events & Alarms** → Use the **Events tab** or **Alarm tab**.  

---

## 📝 Notes

- Works fully **offline** after loading.  
- Sunrise/Sunset values are approximate (formula, no API).  
- Exported settings are JSON-based for portability.
