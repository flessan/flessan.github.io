# ⏰ Clock Studio – Modern Analog Clock

Clock Studio is a **HTML, CSS, and JavaScript** based application that displays a **real-time analog clock** with various additional features.  
It is fully **static** and can run directly in the browser without any backend server.

---

## ✨ Key Features

- **Interactive Analog Clock**  
  Clock hands (hour, minute, second) move in real-time based on the system time.
  
- **Digital Clock & Date**  
  Shows digital time (12/24h format) along with the current date.
  
- **Switchable Themes**  
  4 built-in visual themes:
  - 🌑 Dark  
  - 🌊 Ocean  
  - 🌅 Sunset  
  - 🌳 Forest  

- **Multiple Clock Styles**  
  Choose between `Classic`, `Modern`, `Minimal`, or `Roman Numerals`.

- **Extra Information**  
  - Time zone & UTC offset  
  - Day of the year  
  - Week number  
  - Daily progress (progress bar)  
  - Sunrise & sunset information  

- **World Clock**  
  Displays world times for major cities: New York, London, Tokyo, Sydney.

- **Pomodoro Timer**  
  25-minute timer with **Start/Pause** and **Reset** controls for focus sessions.

- **Sound & Interaction**  
  - Toggle sound on/off  
  - Small sound effect when changing theme or style  

- **Responsive & Modern**  
  Works on both desktop and mobile devices.

---

## 📂 Code Structure

1. **HTML (`<body>`)**
   - **Header**: logo + theme selector buttons.  
   - **Main Content**: split into 2 columns (grid layout)  
     - **Clock Card**: analog + digital clock, controls, style selection.  
     - **Info Card**: tabbed information (Time Info, World Clock, Pomodoro).  
   - **Footer**: short description of the app.

2. **CSS**
   - Uses CSS variables (`:root`) for theme palettes.  
   - Styles for analog clock (hands, numbers, center).  
   - Grid layout, cards, buttons, tabs, progress bars, etc.  
   - Smooth transitions for hover, clicks, and theme changes.

3. **JavaScript (class `ClockApp`)**
   - **updateClock()**: rotates hands + updates digital time.  
   - **updateWorldClocks()**: synchronizes world clocks.  
   - **setClockStyle()**: switches between classic, modern, minimal, roman numeral styles.  
   - **Pomodoro controls**: start, pause, reset timer.  
   - **Event listeners**: theme switcher, style buttons, tab switches, etc.  
   - **Audio context**: simple sound effect system.  

---

## 🚀 How to Run

1. Save the file as `index.html`.  
2. Open it with any modern browser (Chrome, Firefox, Edge, Safari).  
3. The clock will run automatically in real-time.  

---

## 🔧 Customization

- **Default theme**: set `data-theme="ocean"` in the `<html>` tag.  
- **Clock size**: adjust the `--clock-size` variable in CSS.  
- **Hand colors / appearance**: edit variables like `--hour-hand`, `--minute-hand`, etc.  
- **Pomodoro duration**: change `this.pomodoroTime = 25 * 60;` in JavaScript.
