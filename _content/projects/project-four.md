## Overview
**AV Test Studio Pro** is a professional audio-visual testing and diagnostics tool designed to help users test their **camera and microphone functionality** directly in a web browser. The app provides a full-featured interface for testing, recording, and analyzing audio-visual performance with a focus on privacy and user control.

### Key Features
- **Advanced Camera Diagnostics**
  - Real-time preview with support for multiple connected devices.
  - Adjustable video quality settings and digital zoom control.
  - Instant snapshot capture and full video recording with audio.
  - Detailed technical readouts including resolution, frame rate (FPS), and device information.
- **Professional Microphone Analysis**
  - Live audio visualization with switchable modes (frequency bars and waveform oscilloscope).
  - Microphone selector for choosing between available input devices.
  - Audio enhancement toggles for echo control and noise reduction.
  - Integrated 440Hz test tone generator to verify audio output.
- **Real-Time System Monitoring**
  - Displays key system metrics like battery level, memory usage, and current network speed.
  - CPU temperature monitoring (on supported hardware) for a complete performance overview.
- **Privacy-Focused Design**
  - All processing is done locally in the browser; no data is ever sent to external servers.
  - User fully controls camera and microphone access at all times.

### Technologies Used
- MediaDevices API
- Web Audio API
- MediaRecorder API
- Canvas for visualization
- ES6 JavaScript
- Glassmorphism UI with dark mode design
